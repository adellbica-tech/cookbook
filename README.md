                                      家常菜隐私协议

引言
欢迎使用“家常菜”App（以下简称“本应用”）。本应用由[您的姓名/公司名称]（以下简称“我们”）开发并提供服务。我们深知隐私对您的重要性，并承诺保护您在使用本应用时提供的个人信息。
本应用的核心功能是为您提供菜谱查询与烹饪流程参考。请注意：本应用仅需要网络访问权限，用于加载菜谱数据、图片及更新内容。我们不会主动收集、上传或分享您的任何个人身份信息。
请仔细阅读并理解本协议。当您开始使用本应用时，即表示您同意本协议的全部内容。
一、我们如何收集和使用您的个人信息
鉴于本应用的功能设计，我们原则上不收集您的任何个人信息。具体说明如下：
无需注册/登录：您无需提供手机号、邮箱、昵称、密码等任何信息即可直接使用本应用的全部功能。
不收集设备标识符：我们不会读取您的IMEI、MAC地址、Android ID、IDFA等唯一设备标识符。
不收集位置信息：我们不会请求或获取您的地理位置。
不访问存储/相册/通讯录：本应用未声明任何此类权限。
唯一权限：网络访问：该权限仅用于从我们的服务器下载菜谱文本、步骤说明及菜谱图片。我们不会通过网络上传任何关于您或您设备的个人信息。
二、网络访问与数据使用
内容加载：您每次打开菜谱或搜索新菜品时，应用会通过互联网从我们的服务器请求数据。
临时数据：为提升加载速度，应用可能会在您的设备本地（如App缓存目录）临时存储部分菜谱图片或数据。这些数据仅用于优化您的体验，不会被我们远程读取。您可在手机系统设置中清除应用缓存。
第三方服务：本应用目前不集成任何第三方统计分析工具（如友盟、Firebase）、广告SDK或社交分享SDK。因此，不存在通过第三方间接收集您信息的情况。
三、您的权利
由于我们未收集您的个人信息，您可以无需行使“查询、更正、删除个人信息”或“注销账户”等权利。若希望清除本地缓存，请通过手机系统设置删除本应用的缓存文件。
